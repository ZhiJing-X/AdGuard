### EasyList

### EasyPrivacy
从网络上上完全删除所有形式的跟踪，包括Web错误、跟踪脚本和信息收集，从而保护您的个人数据
```bash
https://easylist-downloads.adblockplus.org/easyprivacy.txt
```
